The Problem

善意はあるのに、つながれない

出典：内閣府「市民活動に関する世論調査」

BARRIER 1
参加方法・情報がわからない
助けたい気持ちはあっても、どこで・誰に届ければいいかわからない
BARRIER 2
一緒に活動する仲間がいない
孤独に行動するのは難しく、つながりのきっかけがない
→
batON が解消
SOLUTION 1
フィード & マッチング
スキル・ニーズを投稿し、マッチした相手と即つながれる
SOLUTION 2
DMチャット & 感謝チェーン
マッチ後すぐDMで調整、善意がつながる仕組みを可視化

Implementation

こだわって実装した内容

🃏
MATCHING
カードスワイプ UI
右スワイプでLIKE、左でパス。LIKE/NOPEスタンプ・飛び出しアニメーション・マッチ成立モーダルまで一気通貫で実装。
🌐
3D VISUALIZATION
SceneKit 感謝チェーン
恩送りの伝播をSceneKitで3D可視化。恩人→自分→受益者のつながりをリアルタイムアニメーションで表現。
🎨
DESIGN SYSTEM
統一デザインシステム
バイオレット×シアンのグラデーションを全画面で統一。Color+Extensions.swift で一元管理し、一貫したUXを実現。
🛡️
RELIABILITY
オフラインフォールバック
API接続失敗時もモックデータで動作継続。ハッカソンのデモが必ず動く設計。GraphQLスキーマ駆動でiOS・Backend間の型不一致を防ぐ。
