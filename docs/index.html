Features

主要機能

📰

フィード & 投稿

「助けます」「助けてください」の2種類を投稿。カテゴリ・タグで絞り込み検索、いいね機能も搭載。

❤️

スキルマッチング

Tinder風スワイプUIでユーザーをマッチング。スキルの一致度でスコアリングして推薦。

💬

ダイレクトメッセージ

マッチした相手とDMで詳細調整。リアルタイムメッセージング（既読機能付き）。

🌐

感謝チェーン 3D 可視化

善意がどのように伝播したかを3Dグラフで可視化。自分が起点の恩送りが一目でわかる。

Tech Stack

技術スタック

📱
iOS / SwiftUI
MVVM + ObservableObject、カスタムUIコンポーネント、アニメーション
⚡
GraphQL + Apollo Server
型安全なAPI設計、リゾルバーによる柔軟なデータ取得
🐘
PostgreSQL
UUID主キー、外部キー制約、インデックス最適化で高速クエリ
🐳
Docker
コンテナ化による再現性の高い開発環境・デプロイ
🔐
JWT 認証
bcrypt パスワードハッシュ、トークンベース認証フロー

こだわり

工夫した点

🎨

ダークテーマUI設計

バイオレット×シアンのグラデーションを基調とした独自デザインシステム。Color+Extensions.swift で全画面一貫したスタイルを管理。

🌐

3D 感謝チェーン可視化

SceneKit を使わず Pure SwiftUI で力学グラフアニメーションを実現。善意の波及を直感的に表現。

⚙️

完全型安全 GraphQL

スキーマ駆動開発で iOS 側と Backend 側の型を一致させ、ランタイムエラーを排除。

📱

オフライン対応 UI

API 接続失敗時はモックデータでフォールバック。ハッカソンでも必ずデモが動く設計。
