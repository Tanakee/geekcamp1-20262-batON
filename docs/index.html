Features

主要機能

📰

フィード & 投稿

「恩を送ります」「恩を受けたい」の2種類を投稿。カテゴリ・タグで絞り込み、いいね機能も搭載。

🃏

カードマッチング

カードをスワイプして気になるユーザーにアプローチ。マッチ成立でメッセージ交換へ。

💬

ダイレクトメッセージ

マッチした相手とDMで詳細を調整。リアルタイムメッセージング（既読機能付き）。

🌐

感謝チェーン 3D 可視化

善意がどのように伝播したかをSceneKitの3Dグラフで可視化。恩送りの輪が一目でわかる。

Tech Stack

技術スタック

📱
iOS / SwiftUI
MVVM + ObservableObject、SceneKit 3D、カスタムアニメーション
⚡
GraphQL + Apollo Server
型安全なAPI設計、リゾルバーによる柔軟なデータ取得
🐘
PostgreSQL
UUID主キー、外部キー制約、インデックス最適化で高速クエリ
🐳
Docker
コンテナ化による再現性の高い開発環境・デプロイ
🔐
JWT 認証
bcrypt パスワードハッシュ、トークンベース認証フロー

こだわり

工夫した点

🎨

独自デザインシステム

バイオレット×シアンのグラデーションを全画面で統一。Color+Extensions.swift で一元管理。

🌐

SceneKit 3D 感謝チェーン

善意の伝播をリアルタイム3Dグラフで可視化。ノード・エッジをアニメーションで描画。

⚙️

スキーマ駆動 GraphQL

iOS と Backend の型を一致させるスキーマ駆動開発でランタイムエラーをゼロに。

🛡️

オフラインフォールバック

API 接続失敗時はモックデータで動作継続。ハッカソンのデモが必ず動く設計。
